# Extraer Datos de PDF

Extrae datos estructurados (tablas, campos, valores) de documentos PDF usando IA.

## Endpoint
POST https://pdfrapido.eu/api/extraer-datos

## Input
- PDF (multipart/form-data, campo `file`)

## Output
- Datos estructurados en JSON

## Uso
Útil para procesar facturas, formularios, albaranes u otros documentos con datos tabulares.
