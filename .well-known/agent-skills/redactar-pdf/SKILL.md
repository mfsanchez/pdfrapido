# Redactar PDF

Genera un documento PDF a partir de un prompt o instrucciones en lenguaje natural.

## Endpoint
POST https://pdfrapido.eu/api/redactar-pdf

## Input
- Instrucciones o contenido en texto (campo `prompt`)

## Output
- PDF generado

## Uso
Útil para crear cartas, informes, contratos u otros documentos de texto.
