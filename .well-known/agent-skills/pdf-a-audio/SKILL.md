# PDF a Audio

Convierte el texto de un PDF en audio (text-to-speech) usando IA.

## Endpoint
POST https://pdfrapido.eu/api/pdf-a-audio

## Input
- PDF (multipart/form-data, campo `file`)

## Output
- Archivo de audio MP3

## Uso
Útil para escuchar documentos, libros o informes mientras se realiza otra actividad.
