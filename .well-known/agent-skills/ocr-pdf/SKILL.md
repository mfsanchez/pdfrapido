# OCR a PDF

Extrae texto de PDFs escaneados o imágenes usando reconocimiento óptico de caracteres (OCR).

## Endpoint
POST https://pdfrapido.eu/api/ocr-a-pdf

## Input
- PDF escaneado (multipart/form-data, campo `file`)

## Output
- PDF con texto seleccionable incrustado

## Uso
Útil para hacer buscables documentos escaneados, facturas, contratos u otros PDFs sin texto digital.
