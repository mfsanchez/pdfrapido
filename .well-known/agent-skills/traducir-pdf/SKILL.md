# Traducir PDF

Traduce el contenido de un PDF a otro idioma manteniendo el formato original.

## Endpoint
POST https://pdfrapido.eu/api/traducir-pdf

## Input
- PDF (multipart/form-data, campo `file`)
- Idioma destino (campo `idioma`)

## Output
- PDF traducido

## Uso
Útil para traducir documentos técnicos, contratos o informes a otro idioma.
