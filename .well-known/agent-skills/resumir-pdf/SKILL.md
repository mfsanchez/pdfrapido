# Resumir PDF

Resume el contenido de un documento PDF usando inteligencia artificial.

## Endpoint
POST https://pdfrapido.eu/api/resumir-pdf

## Input
- PDF (multipart/form-data, campo `file`)

## Output
- Resumen en texto del documento

## Uso
Útil para extraer los puntos clave de documentos largos: informes, artículos académicos, contratos.
