# WebMCP — Herramientas PDFRápido en el Navegador

Expone las herramientas de PDFRápido a agentes de IA mediante la API WebMCP (`navigator.modelContext`).

## Tecnología
WebMCP (navigator.modelContext.provideContext) — disponible en navegadores compatibles.

## Herramientas expuestas
- `pdfrapido_ocr` — Extrae texto de PDFs escaneados
- `pdfrapido_resumir` — Resume documentos con IA
- `pdfrapido_chat` — Responde preguntas sobre PDFs
- `pdfrapido_traducir` — Traduce PDFs a otro idioma
- `pdfrapido_extraer_datos` — Extrae datos estructurados
- `pdfrapido_corregir` — Corrige gramática y ortografía
- `pdfrapido_redactar` — Genera PDFs desde texto
- `pdfrapido_pdf_a_audio` — Convierte PDF en audio MP3
- `pdfrapido_pdf_a_imagen` — Convierte PDF en imágenes PNG
- `pdfrapido_comprimir` — Comprime PDFs en el navegador
