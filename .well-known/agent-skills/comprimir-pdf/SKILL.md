# Comprimir PDF

Reduce el tamaño de un archivo PDF procesándolo completamente en el navegador (sin subir a servidores).

## Herramienta
https://pdfrapido.eu/comprimir-pdf

## Tecnología
WebAssembly — el archivo nunca abandona el dispositivo del usuario.

## Uso
Útil para reducir el peso de PDFs antes de enviarlos por email o subirlos a plataformas con límite de tamaño.
