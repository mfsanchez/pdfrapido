# PDF a Imagen

Convierte páginas de un PDF en imágenes PNG de alta calidad.

## Endpoint
POST https://pdfrapido.eu/api/pdf-a-png

## Input
- PDF (multipart/form-data, campo `file`)

## Output
- Imágenes PNG (una por página)

## Uso
Útil para previsualizar PDFs, extraer páginas como imágenes o insertar páginas en presentaciones.
