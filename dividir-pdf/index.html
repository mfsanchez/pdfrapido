Elige las páginas viendo el documento

Al cargar el PDF verás una miniatura de cada página. Haz clic en las que quieres extraer y se irán marcando; o escribe el rango a mano (por ejemplo 1-3, 5, 8-10). También puedes arrastrar las miniaturas para reordenarlas antes de dividir, y el PDF final respetará ese orden. Todo se renderiza en tu propio navegador, así que el archivo no sale de tu dispositivo en ningún momento.

Dividir un PDF en 3 pasos

1

Carga el PDF

Arrástralo o selecciónalo; verás una miniatura de cada página.

2

Elige qué extraer

Haz clic en las páginas que quieres (modo Extraer rango) o elige Separar todas.

3

Descarga

Obtienes un PDF con tu selección, o un archivo independiente por cada página.

Para qué sirve: situaciones reales

Enviar solo lo necesarioManda únicamente las páginas firmadas de un contrato, no las 40 completas.
Sacar un capítulo o temaExtrae una sección concreta de un libro o de unos apuntes en PDF.
Separar un escaneado múltipleConvierte un PDF con DNI, nómina y recibo escaneados juntos en archivos sueltos.
Quedarte con unas facturasDe un PDF que junta las facturas de un proveedor, extrae solo las que necesitas.
Aislar una página para firmarSaca una página para firmarla o sellarla y vuelve a unirla luego con Unir PDF.

Errores comunes y cómo evitarlos

  • «Introduce un rango de páginas válido». El rango está vacío o fuera de límites: usa números reales del documento (por ejemplo 1-3, 5).
  • «Error al dividir el PDF». Suele ser un PDF protegido con contraseña; desbloquéalo antes con Desbloquear PDF.
  • Elegí «Separar todas» y el navegador bloqueó las descargas. Ese modo genera un archivo por página y se descargan en cadena: permite las descargas múltiples de este sitio.
  • Quiero otro orden. Arrastra las miniaturas a la posición deseada antes de pulsar dividir.

Preguntas frecuentes sobre dividir PDF

¿Puedo elegir exactamente qué páginas extraer?

Sí. Haz clic en las miniaturas de las páginas que quieres, o escribe un rango de texto como 1-3, 5, 8-10. Solo esas páginas irán al PDF resultante.

¿Se mantiene el texto del PDF?

Sí. Las páginas se copian tal cual, así que el texto del PDF resultante sigue siendo seleccionable y las imágenes conservan su calidad.

¿Puedo separar todas las páginas en archivos sueltos?

Sí. Con el modo Separar todas obtienes un PDF independiente por cada página del documento.

¿Mis archivos se suben a un servidor?

No. Todo el proceso ocurre en tu navegador; el PDF no sale de tu dispositivo. Puedes comprobarlo poniéndote en modo avión.

¿Puedo reordenar las páginas antes de dividir?

Sí. Arrastra las miniaturas para cambiar el orden; el PDF resultante respetará el orden que dejes.

¿Puedo dividir un PDF con contraseña?

No directamente: un PDF cifrado dará error. Quítale la contraseña antes en Desbloquear PDF y vuelve a intentarlo.
