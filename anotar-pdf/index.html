Cómo anotar un PDF online

Anotar documentos PDF es una tarea habitual al revisar contratos, corregir trabajos, tomar apuntes de estudio o marcar planos de diseño. Con esta herramienta puedes hacerlo directamente en el navegador, sin instalar ningún software.

1

Sube tu PDF

Arrastra el archivo o haz clic para seleccionarlo desde tu dispositivo.

2

Añade anotaciones

Elige una herramienta, selecciona color y grosor, y marca el documento.

3

Descarga el resultado

Pulsa «Guardar PDF anotado» para descargar el PDF con todas tus marcas.

Herramientas de anotación disponibles

La barra lateral incluye diez herramientas para cubrir cualquier necesidad de marcado: resaltado para colorear fragmentos de texto, subrayado para enfatizar líneas, comentarios en forma de nota adhesiva, texto libre, dibujo a mano alzada, flechas para señalar elementos, rectángulos y elipses para enmarcar zonas, y la herramienta borrar para eliminar cualquier anotación.

Privacidad total: tu PDF no sale del navegador

A diferencia de otros servicios online, esta herramienta procesa el PDF íntegramente en tu dispositivo. El archivo nunca se sube a ningún servidor, lo que garantiza la confidencialidad de contratos, facturas, datos personales o cualquier otro documento sensible.

Preguntas frecuentes

¿Puedo anotar un PDF sin instalar nada?
Sí. La herramienta funciona completamente en el navegador: no requiere instalar software, registrarse ni enviar el archivo a ningún servidor.
¿Las anotaciones se guardan en el PDF definitivamente?
Al pulsar «Guardar PDF anotado» se descarga un nuevo PDF con las anotaciones incrustadas como imágenes, listas para compartir o imprimir.
¿Qué tipos de anotaciones puedo añadir?
Puedes subrayar texto, añadir notas adhesivas, escribir texto libre, dibujar a mano alzada, insertar flechas, rectángulos y elipses, y borrar cualquier anotación con un solo clic.
