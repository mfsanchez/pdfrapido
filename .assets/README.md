# PDFRápido

**Herramientas PDF Online Gratis — 30+ utilidades sin registro**

PDFRápido es una plataforma web gratuita que permite trabajar con archivos PDF directamente desde el navegador. Une, divide, comprime, convierte, edita, firma y protege PDFs sin instalar software ni crear cuentas.

## 🚀 Características principales

- **30+ herramientas PDF** gratuitas
- **Sin registro** ni suscripciones ocultas
- **Procesamiento seguro**: los archivos se eliminan automáticamente
- **Multiplataforma**: funciona en Windows, Mac, Linux, iOS y Android
- **Herramientas con IA**: resumir, traducir, analizar contratos y extraer datos
- **100% en español**: interfaz optimizada para usuarios hispanohablantes

## 🛠 Stack técnico

- Frontend: HTML5, CSS3, JavaScript vanilla
- Backend: PHP, Python
- Servidor: Caddy
- Procesamiento PDF: WebAssembly, Python (PyMuPDF, pdf2image)
- IA: Groq API (Llama 3.3)
